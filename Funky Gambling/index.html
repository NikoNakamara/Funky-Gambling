Quick setup

  1. Invite Funky Gambling to your server.
  2. Run fumo setup as a moderator.
  3. Pick popup, filter & gambling settings.
  4. Enjoy your new cozy bot ✨

Main features

Leveling system

Members gain XP by chatting and receive cute level-up popups in the channel you choose. You can disable popups per-server with fumo level popup false.

Gambling economy

Simple coin-flip style gambling with server-wide balance. Use fumo cf, daily rewards, and more — and lock it to a specific channel.

Bad-word filter

Built-in insult filter that automatically removes messages with slurs or harsh phrases. Edit the global list with fumo cwl or turn it off if you don't need it.

Guided setup

Run fumo setup to walk through level popup channel, filter toggle, gambling toggle, and gambling channel — all via buttons & dropdowns.
