Commands

Prefix: fumo

Setup & configuration

fumo setup

Interactive setup wizard. Lets you configure level popup channel, filtered words toggle, gambling toggle and gambling channel through buttons & dropdowns.

fumo cwl

Shows the current global and server extra cancelled words list. Moderators can open a popup to edit the global lists.

fumo cwl true / false

Enable or disable the bad-word filter for this server.

fumo level popup true / false

Turn level-up popup messages on or off for this server.

Leveling

fumo level

Shows your current level and XP progress.

fumo level @user

Shows level and XP for another member.

Economy & gambling

fumo bal

Shows your current token balance.

fumo daily

Claim a daily reward of tokens.

fumo cf <amount/all/half>

Coin flip gambling. Win or lose the amount you bet.

Help

fumo help

Shows an in-Discord help menu for Funky Gambling
